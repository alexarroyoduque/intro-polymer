Variables computadas

Propiedades computadas

Podemos crear funciones que interpretan el valor de sus argumentos y devuelven un resultado.

computed: 'computeFullName(name, lastname)'

Atributos dinámicos

Es posible usar variables en atributos para que dependiendo de su valor reciban un valor u otro.

Es importante considerar que en atributos nativos de HTML es necesario agrega un $ antes del =

class$="{{suit}}"

intro-hero.html

<dom-module id="intro-hero">
  <template>
    <p class$="{{suit}}">El hombre del mañana: 
      <b>{{hero}}</b>
    </p>
    <button on-click="changeClothes">Cambiarse de ropa</button>
  </template>
  <style>
    .superman {
      background-color: yellow;
      color: red;
      border: 2px solid blue;
      padding: 5px;
    }
  </style>
  <script>
    Polymer({
      is: "intro-hero",
      properties: {
        hero: {
          type: String,
          computed: 'computeFullName(name, lastname)'
        },
        name: {
          type: String,
          value: 'Clark'
        },
        lastname: {
          type: String,
          value: 'Kent'
        },
        suit: {
          type: String
        }
      },
      computeFullName: function(name, lastname) {
        return name + ' ' + lastname;
      },
      changeToSuperMan: function () {
        this.name = 'Super Man';
        this.lastname = '';
        this.suit = 'superman';
      },
      changeToClark: function () {
        this.name = 'Clark';
        this.lastname = 'Kent';
        this.suit = '';
      },
      changeClothes: function () {
        if (this.suit === 'superman') {
          this.changeToClark();
        } else {
          this.changeToSuperMan();
        }
      }
    });
  </script>
</dom-module>
