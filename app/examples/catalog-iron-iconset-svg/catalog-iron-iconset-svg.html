Catálogo: iron-iconset-svg

Polymer proporciona una colección de elementos en su catálogo.

iron-iconset-svg

Para usar el iron-iconset-svg del catálogo de elementos Polymer es necesario instalar las dependencias necesarias e importarlas al proyecto.

bower install --save PolymerElements/iron-iconset-svg

Podemos definir nuestros iconos personalizados con SVG.

En lugar de importar el set completo de iconos podemos usar esta técnica para importar solo los iconos necesarios para el proyecto. Así mejoramos el rendimiento aligeran la carga de la página.

Más información en este vídeo.

Ejemplo

index.html

Es necesario importar los elementos Polymer con los iconos que queremos usar.

<script src="../../bower_components/webcomponentsjs/webcomponents-lite.js"></script>
<link rel="import" href="../../bower_components/iron-icons/iron-icons.html">
<link rel="import" href="./custom-icons.html">

Usando los iconos

  <iron-icon icon="custom-icons:android"></iron-icon>
