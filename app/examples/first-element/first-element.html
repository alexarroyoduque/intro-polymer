Mi primer elemento

¿Qué necesito?

Polymer

En el bower.json es necesario incluir Polymer como dependencia:

  "dependencies": {
    "polymer": "Polymer/polymer#1.0.0"
  }

Creando un elemento

Nuestro componente necesita importar polymer.html.

El componente se crea dentro de la etiqueta dom-module.

La etiqueta template contiene el HTML del componente.

El nombre de los componentes siempre necesita al menos un guión medio -.

Crearemos un tag script para ejecutar Polymer y crear un componente.

intro-hero.html

  <link rel="import"
      href="../../bower_components/polymer/polymer.html">

  <dom-module id="intro-hero">

    <template>
      <span>Superman</span>
    </template>

    <style>
      span {
        background-color: yellow;
        color: red;
        border: 2px solid blue;
        padding: 5px;
      }
    </style>

    <script>
      Polymer({
        is: "intro-hero"
      });
    </script>

  </dom-module>

Usando mi elemento

Para usar el componente en un HTML es necesario importarlo.

index.html

  <!DOCTYPE html>
  <html>
    <head>
      <meta charset="UTF-8">
      <title>Intro</title>
      <script
        src="../../bower_components/webcomponentsjs/webcomponents-lite.js"></script>
      <link rel="import" href="./intro-hero.html">
    </head>

    <body>
      <intro-hero></intro-hero>
    </body>
  </html>
