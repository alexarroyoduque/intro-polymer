Ciclo de vida

El componente intro-krypton pasa por las siguientes fases:

1. created
El componente es creado.
2. ready
El componente está listo para ser usado. Polymer ha terminado de crear e inicializar el DOM local del componente.
3. attached
El componente ha sido agregado al DOM de la página. Dentro de esta fase es posible acceder de forma asíncrona a elementos padre o hermanos del componente.

intro-krypton.html

  <link rel="import"
      href="../../bower_components/polymer/polymer.html">

  <dom-module id="intro-krypton">
    <template>
      <div><span>Krypton</span></div>
    </template>

    <style>
      div {
        position: relative;
        color: white;
        background-color: #4CAF50;
        padding: 5px;
        height: 200px;
        width: 200px;
        border-radius: 50%;
      }

      span {
        position: absolute;
        top: 42%;
        right: 31%;
        font-size: 24px;
      }
    </style>

    <script>
      Polymer({
        is: "intro-krypton",
        created: function () {
          console.log('created');
        },
        ready: function () {
          console.log('ready');
        },
        attached: function () {
          console.log('attached');
        },
        detached: function () {
          console.log('detached');
        }
      });
    </script>
  </dom-module>
