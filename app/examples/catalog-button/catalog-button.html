Catálogo: botón

Polymer proporciona una colección de elementos en su catálogo.

Botón

Para usar el botón del catálogo de elementos Polymer es necesario instalar las dependencias necesarias e importarlas al proyecto.

bower install --save PolymerElements/paper-button

index.html

flat button raised button No ripple effect toggle-able button 
<html>
  <head>
  <script
    src="../../bower_components/webcomponentsjs/webcomponents-lite.js"></script>
  <link
    rel="import" href="../../bower_components/paper-button/paper-button.html">
  </head>
  <body>
    <div>
      <paper-button>flat button</paper-button>
      <paper-button raised>raised button</paper-button>
      <paper-button noink>No ripple effect</paper-button>
      <paper-button toggles>toggle-able button</paper-button>
    </div>
  </body>
</html>
